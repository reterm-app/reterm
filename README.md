## What is Reterm?

Reterm is a desktop and mobile application that brings all your server connections into one place. Instead of juggling multiple terminal windows, SSH configs, and FTP clients, you get a single unified interface — with tabs, just like your browser.

Whether you're a seasoned sysadmin managing dozens of servers or a developer connecting to a remote machine for the first time, Reterm is built for you.

> **The terminal doesn't have to be scary.** Reterm's built-in AI assistant guides you through server management tasks step by step, so you can focus on getting things done instead of googling commands.

## Features

### Protocols
| Protocol | Status |
|---|---|
| **SSH** — Secure Shell | ✅ Planned |
| **SFTP** — SSH File Transfer Protocol | ✅ Planned |
| **FTP** — File Transfer Protocol | ✅ Planned |
| **RDP** — Remote Desktop Protocol | ✅ Planned |

### Core
- **Browser-like tabs** — open and switch between multiple live connections simultaneously, just like browser tabs
- **Connection manager** — organize servers into groups, tag them, and find them instantly
- **Cloud sync** — your connections are saved and synced across all your devices automatically
- **End-to-end encryption** — all connection data stored in the cloud is E2E encrypted; your credentials never leave your device unencrypted

### AI Assistant
- Built-in assistant that explains commands, suggests fixes, and guides beginners through complex tasks
- Helps new users go from zero to productive without needing to know every flag by heart
- Available in the premium tier
